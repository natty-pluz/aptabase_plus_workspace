# aptabase_storage_hive_ce_plus

`aptabase_storage_hive_ce_plus` provides a Hive CE storage adapter for queued Aptabase Plus events.

Use this package when you want persistent queue storage that works in pure Dart and Flutter runtimes.

## Why Hive CE

Hive CE is a lightweight key-value database written in pure Dart. It supports mobile, desktop, and browser environments, has no native dependencies, and is a strong fit for an offline analytics queue.

This adapter stores each queued Aptabase event as a string value in a Hive box. The event key is generated by `aptabase_plus`, and the adapter reads keys with the configured prefix.

## Installation

```yaml
dependencies:
  aptabase_plus:
    path: ../aptabase_plus
  aptabase_storage_hive_ce_plus:
    path: ../aptabase_storage_hive_ce_plus
```

For Flutter apps, pair it with `aptabase_flutter_plus`:

```yaml
dependencies:
  aptabase_flutter_plus:
    path: ../aptabase_flutter_plus
  aptabase_storage_hive_ce_plus:
    path: ../aptabase_storage_hive_ce_plus
```

When published, replace path dependencies with published version constraints.

## Dart usage

```dart
import 'package:aptabase_plus/aptabase_plus.dart';
import 'package:aptabase_storage_hive_ce_plus/aptabase_storage_hive_ce_plus.dart';

Future<void> main() async {
  final storage = await HiveCeAptabaseStorage.open(
    directoryPath: '.dart_tool/aptabase_events',
  );

  final client = await Aptabase.init(
    'A-US-0000000000',
    storage: storage,
    systemInfoProvider: const StaticAptabaseSystemInfoProvider(
      AptabaseSystemInfo(
        osName: 'dart',
        osVersion: '3.x',
        locale: 'en',
        appVersion: '1.0.0',
        appBuildNumber: '1',
        isDebug: false,
      ),
    ),
  );

  await client.trackEvent('cli_started');
  await client.flush();
  await Aptabase.dispose();
  await storage.close();
}
```

## Flutter usage

```dart
import 'package:aptabase_flutter_plus/aptabase_flutter_plus.dart';
import 'package:aptabase_storage_hive_ce_plus/aptabase_storage_hive_ce_plus.dart';

final storage = await HiveCeAptabaseStorage.open(
  boxName: 'aptabase_events',
);

await Aptabase.init(
  'A-US-0000000000',
  storage: storage,
);
```

In a Flutter app that already initialises Hive globally, omit `directoryPath`. In a Dart CLI/test environment, pass `directoryPath` so Hive knows where to store the box.

## Configuration

### Box name

```dart
final storage = await HiveCeAptabaseStorage.open(
  boxName: 'my_app_aptabase_events',
);
```

### Key prefix

```dart
final storage = await HiveCeAptabaseStorage.open(
  keyPrefix: 'aptabase_event_',
);
```

The adapter only returns entries whose keys start with `keyPrefix`, so the box can safely contain unrelated values if needed.

### Encryption

Hive CE supports encrypted boxes. The adapter exposes both `encryptionCipher` and `encryptionKey` so an app can opt into Hive-supported encryption when appropriate.

```dart
final storage = await HiveCeAptabaseStorage.open(
  encryptionKey: myGeneratedKey,
);
```

Key generation, storage, and rotation should be handled by the application.

## Behaviour

- `open()` opens the Hive box and returns the adapter.
- `init()` is a no-op because the box is already open.
- `addEvent()` writes the event payload by key.
- `getItems()` reads prefixed keys in sorted order and returns up to the requested batch length.
- `deleteEvents()` deletes events after a successful send or a non-retryable discard.
- `close()` flushes pending Hive writes and closes the box.

## Shared preferences vs Hive CE

| Adapter | Best for | Notes |
| --- | --- | --- |
| `SharedPreferencesAptabaseStorage` | Simple Flutter apps | Small setup, Flutter plugin dependency, simple key-value persistence. |
| `HiveCeAptabaseStorage` | Dart/Flutter apps with stronger local queue needs | Pure Dart database, better fit for heavier queues, optional Hive encryption support. |

## Testing

Tests create a temporary directory and open real Hive boxes there:

```bash
dart test packages/aptabase_storage_hive_ce_plus
```

The test structure mirrors the source structure:

```text
lib/src/hive_ce_aptabase_storage.dart
test/src/hive_ce_aptabase_storage_test.dart
```
